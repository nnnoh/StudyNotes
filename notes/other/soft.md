## IDAE

#### 设置鼠标放到类或方法上自动显示文档

File -> Settings -> Editor -> General

勾选 Other -> show quick documentation...

#### 自定义代码折叠

```java
// <editor-fold defaultstate="collapsed" desc="$description$">
$SELECTION$
// </editor-fold>
```

#### 查看类继承关系

顶部菜单栏 Navigate -> Type Hierachy

#### 查看使用

右键菜单 -> Find Usage
